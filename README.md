#Technical Project
Overview

This is a Java Spring Boot project showcasing my ability to implement modern backend functionalities, including authentication, API integration, email handling, and data processing.

The project is built to demonstrate coding skills rather than solving a real-world problem.

Features

Authentication

JWT (JSON Web Token) authentication

Basic authentication support

User Management

User entity to manage users

Hardcoded user for demonstration purposes

Weather Data Integration

Fetches weather data from an external API

Converts weather data to a string format

Email Broadcasting

Sends weather updates to the hardcoded user

Demonstrates email sending through JavaMailSender

Database Integration

User information stored in a relational database (e.g., PostgreSQL)

Tracks email broadcasts

Technologies Used

Java 21

Spring Boot

Spring Security (JWT & Basic Auth)

Spring Data JPA

PostgreSQL (or H2 for testing)

JavaMailSender for sending emails

External Weather API integration
